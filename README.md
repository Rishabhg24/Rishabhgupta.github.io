# Rishabh Gupta Portfolio

A modern, responsive portfolio website showcasing my work as a Python Backend Developer.

## Features

- **Modern Design**: Clean, professional interface with glass morphism effects and smooth animations
- **Responsive Layout**: Fully responsive design that works on all devices
- **Interactive Elements**: Typing animation, hover effects, and smooth scrolling
- **Performance Optimized**: Minified CSS/JS, optimized images, and fast loading
- **SEO Friendly**: Proper meta tags and semantic HTML structure

## Technology Stack

- **Frontend**: HTML5, CSS3, JavaScript, Bootstrap 5
- **Animations**: AOS (Animate On Scroll), Owl Carousel
- **Icons**: Ionicons, Font Awesome
- **Optimization**: Minified assets, optimized images

## Sections

- **Home**: Hero section with animated typing effect
- **About**: Professional summary and skills showcase
- **Education**: Academic background
- **Projects**: Portfolio of web development projects
- **Contact**: Contact information and social links

## Getting Started

1. Clone the repository
2. Open `index.html` in your browser
3. No build process required - it's a static website

## Customization

- Edit `css/custom-enhancements.css` for styling changes
- Update `js/typing-animation.js` for hero section text
- Modify `index.html` for content changes

## Deployment

This site is optimized for GitHub Pages deployment. Simply push to the main branch and enable GitHub Pages in repository settings.
